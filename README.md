# 🛒 Whipped

Whipped는 친환경 제품을 판매하는 쇼핑몰 웹사이트입니다. 이 프로젝트는 React와 Express를 사용하여 개발되었습니다.

## 🚀 기능

- 제품 목록 보기
- 장바구니에 제품 추가 및 제거
- 주문 및 결제 기능
